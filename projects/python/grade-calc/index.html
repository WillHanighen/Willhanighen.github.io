Grade Calculator

A Python application to calculate your grade percentage based on points earned and total possible points.

Project Details

About This Project

This is a simple grade calculator built in Python. It allows you to:

  • Calculate your grade percentage based on points earned and total possible points
  • See your score formatted with proper precision
  • Calculate multiple grades in a single session
